Markdown 基础语法

一、标题

1. 使用#标记标题级别

使用#号可表示1-6级标题，一级标题对应一个#号，二级标题对应两个#号，以此类推。
注：#和「标题」之间建议保留一个字符的空格，这是最标准的MarkDown写法。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

二、段落

段落没有特殊的格式，直接编写文字就好，段
落的换行是使用两个以上空格加上回车。

也可以在段落后面使用一个空行来表示重新开始一个段落。

5. 脚注

脚注是对文本的补充说明。
格式:

[^要注明的文本key]
	[^要注明的文本key]:说明

脚注的用法：

创建脚注格式类似这样[^Markdown]。

三、列表

Markdown 支持有序列表和无序列表。

无序列表

使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容：

	+ 文本一
		+ 文本二
			- 文本三
  • 文本一
    • 文本二
      • 文本三

有序列表

在文字前面加上数字并加上.就可以了，例如：

1. 文本一
2. 文本二
3. 文本三
  1. 文本一
  2. 文本二
  3. 文本三

列表嵌套

列表嵌套只需在子列表中的选项前面添加制表符或四个空格即可：

	+ 文本一
		+ 文本二
			- 文本三
  • 文本一
    • 文本二
      • 文本三

六、链接

格式：
[链接名称](链接地址)或<链接地址>

例如：
这是一个链接 [百度](https://baidu.com)
这是一个链接 百度

<https://baidu.com>
https://baidu.com

高级链接

可以通过变量来设置一个链接，变量赋值在文档末尾进行：
格式：

这个链接用 key 作为网址变量 [text][key]  
然后在文档的结尾为变量赋值（网址）
	[key]: http://www.xxx.com/ "title(可选)"
key 可以是字符数字

推荐网站，[B][key] [A][2:,] [T][tencent]

	[key]: https://www.baidu.com/ "百度"
	[2:,]: https://www.alibaba.com/ "阿里巴巴"
	[tencent]: https://www.tencent.com/ "腾讯"

推荐网站，B A T

七、图片

  • 图片语法格式：
![alt 属性文本](图片地址)
![alt 属性文本](图片地址 "可选标题（title属性）")

例如：
![baidu](https://www.baidu.com/img/flexible/logo/pc/result.png)

  • 也可以像网址那样对图片网址使用变量:
    例如：
![baidu logo][logourl]
	[logourl]:https://www.baidu.com/img/flexible/logo/pc/result.png "百度logo"

![baidu logo][logourl]
[logourl]:https://www.baidu.com/img/flexible/logo/pc/result.png "百度logo"

  • 结合链接可以将图片组合成可点击链接：
    [![alt 属性文本](图片地址 "可选标题（title属性）")](url)

  • Markdown无法指定图片的高度与宽度，可以使用普通的<img>标签

八、表格

表格使用|来分隔不同的单元格，使用-来分隔表头和其他行。
语法格式如下：

| 左对齐标题 | 右对齐标题 | 居中对齐标题 |
| :------| ------: | :------: |
|000000000000000| 000000000000001 | 000000000000002|
|10|11|
左对齐标题 右对齐标题 居中对齐标题
000000000000000 000000000000001 000000000000002
10 11
  • 对齐方式(默认居中对齐)：
    • -: 设置内容和标题栏居右对齐。
    • :- 设置内容和标题栏居左对齐。
    • :-: 设置内容和标题栏居中对齐。

九、其他

1. 支持的 HTML 元素

不在Markdown涵盖范围之内的标签，都可以直接在文档里面用HTML撰写。
目前支持的HTML元素有：<kbd> <b> <i> <em> <sup> <sub> <br>等 ，如：

使用 <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>Del</kbd> 重启电脑
使用 Ctrl+Alt+Del 重启电脑

2. 转义

\`*_{}[]()#+-.!

3. 公式

Markdown Preview Enhanced使用 KaTeX 或者 MathJax 来渲染数学表达式。

KaTeX 拥有比 MathJax 更快的性能，但是它却少了很多 MathJax 拥有的特性。可以查看 KaTeX supported functions/symbols 来了解 KaTeX 支持那些符号和函数。

默认下的分隔符：

$...$ 或者 \(...\) 中的数学表达式将会在行内显示。  
$$...$$ 或者 \[...\] 或者 ```math 中的数学表达式将会在块内显示。

4. 锚点

Markdown的语法是不支持锚点的，可以使用HTML的锚点语法，在页面中点击一个链接跳转到页面中锚点所在的位置。

建立一个跳转的链连接，这个语法和普通链接的语法一样，区别就是括号内的链接以#起始。

[文本](#idValue)

[跳转](#idValue)
<span id="idValue">跳到这里</span>
